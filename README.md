# React Shopping Cart

**Shopping Cart Project** built with **React JS**, using **Context API** & **useReducer Hook**.


## Features:

- #### Add to Cart
- #### Remove from Cart
- #### Increment / Decrement the Quantity



## Technologies used:

- #### **React JS**
- #### **SASS**


## Available Scripts:

#### `npm start`

#### `npm test`

#### `npm run build`

#### `npm run eject`
